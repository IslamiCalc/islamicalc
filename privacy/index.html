✅

خلاصة: بياناتك تبقى على جهازك فقط. لا نبيع بياناتك. لا نشاركها مع أطراف ثالثة للتسويق.

1️⃣ ما البيانات التي نجمعها؟

  • الموقع الجغرافي — لأوقات الصلاة واتجاه القبلة فقط (بإذنك)
  • اسم المدينة — يُرسل لـ Aladhan API لجلب أوقات الصلاة
  • عند التسجيل الاختياري — البريد الإلكتروني واسم المستخدم عبر Firebase
  • بيانات الاستخدام المجهولة — عدد الزيارات بدون هوية شخصية

2️⃣ البيانات المحلية (localStorage)

بياناتك (ختمة القرآن، XP الأرينا، الأذكار، إعدادات الزكاة) تُحفظ على جهازك فقط. لا ترسل لخوادمنا. يمكنك حذفها من إعدادات المتصفح في أي وقت.

3️⃣ الخدمات الخارجية

Aladhan API أوقات الصلاة — لا يجمع بيانات شخصية
Firebase (Google) تسجيل الدخول الاختياري فقط
Google Fonts خط Cairo — لا بيانات شخصية

4️⃣ حقوقك

  • حق الوصول لبياناتك وتصحيحها أو حذفها
  • حق سحب الموافقة على الموقع الجغرافي في أي وقت
  • حق حذف حسابك ومزامنة بياناتك نهائياً

5️⃣ التواصل

لأي استفسار: privacy@islamicalc.com أو عبر صفحة التواصل.
